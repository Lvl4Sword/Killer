# Development
## Linux
```bash
git clone https://github.com/Lvl4Sword/Killer.git
cd Killer
mkdir -p ~/.virtualenvs/
python3 -m venv ~/.virtualenvs/killer
source ~/.virtualenvs/killer/bin/activate
pip install -U -r requirements.txt
pip install -U -r dev-requirements.txt
```
